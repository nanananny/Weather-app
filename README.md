#  Weather App

A simple weather app using a public API to fetch real-time weather data.

##  Features
- Search for any city
- Live weather updates
- Temperature, humidity, description
- API integration practice project

##  Tech Stack
- HTML
- CSS
- JavaScript (Fetch API)
